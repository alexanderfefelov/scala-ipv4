Scala IPv4 0.1 API

This document is the API specification for Scala IPv4 0.1 API

Class Summary
class IpAddress (val value : Long) extends be.jvb.iptypes.SmallByteArray
Represents an IPv4 address.
class IpAddressPool (val override first : IpAddress, val override last : IpAddress, val freeRanges : scala.collection.immutable.SortedSet[IpAddressRange]) extends IpAddressRange
Represents a pool of IPv4 addresses. A pool is a range of addresses, from which some can be "allocated" and some can be "free".
class IpAddressRange (val first : IpAddress, val last : IpAddress) extends scala.Ordered[IpAddressRange]
Represents a continuous range of IPv4 ip addresses (bounds included).
class IpNetwork (val address : IpAddress, val mask : IpNetworkMask) extends IpAddressRange
Represents an Ipv4 network (i.e. an address and a mask).
class IpNetworkMask (val override value : Long) extends IpAddress
Represents an IPv4 network mask.
class MacAddress (val value : Long) extends be.jvb.iptypes.SmallByteArray
Represents a MAC address.
